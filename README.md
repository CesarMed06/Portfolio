# César Medina — Personal Portfolio

Personal web portfolio built with **React + Vite**. Dark design with an orange/fire-red palette, smooth page transitions and animations powered by **GSAP**.

🌐 **Live:** [portfolio-phi-livid-75.vercel.app](https://portfolio-phi-livid-75.vercel.app/)

---

## Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React 19, React Router DOM v7 |
| Animations | GSAP 3 + ScrollTrigger |
| Styles | Modular CSS with custom properties |
| Build | Vite 8 |
| Testing | Vitest + Testing Library |
| Icons | Simple Icons CDN + inline SVG |

## Pages

| Route | Description |
|---|---|
| `/` | Animated hero, live stats counter and featured projects slider |
| `/trayectoria` | Timeline of professional experience and academic background |
| `/proyectos` | Project catalog with tech tags and GitHub links |
| `/tecnologias` | Full stack organized by category |

## Features

- 🌍 Bilingual — Spanish / English toggle
- 🖱️ Custom animated cursor
- 📱 Fully responsive — mobile first
- ♿ Accessible — semantic HTML, keyboard navigation, reduced motion support
- 🔍 SEO ready — meta tags, Open Graph, dynamic page titles, lang attribute
